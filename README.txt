
# Site Portails – Statique

Ce pack contient un site statique prêt à déployer (Emergent.sh, Netlify, Vercel Static, OVH, etc.).
## Remplacements rapides
- **Nom / SIRET / Téléphone** : dans `index.html` et les modèles (rechercher `06 00 00 00 00` et `★★ Ta Société Portails ★★`).
- **Zone** : remplacer `Île-de-France` par tes villes cibles.
- **Formulaire** : remplace le `onsubmit` par ton outil (Netlify Forms, Tally, Typeform, n8n).

## Déploiement
- Emergent.sh : crée un projet “Static site” et importe ce dossier / le ZIP.
- Ou dépose le dossier sur ton hébergeur (www/).

Bon lancement !
